Get payment using your paytm QR code on your website.

== Description ==
This plugin allow you to accept payments using Paytm QR code. User can place order on your site and can pay using paytm. Simplest way to collect payment directly into your paytm wallet and gain user engagements with easy checkout.

== Installation ==
1. Upload paytm-qr-gateway folder to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Goto woocommerce->settings->checkout->paytm QR 
4. Provide required information and enable paytm QR gateway
5. Please upload your paytm QR code image in your Wordpress image gallery and provide link in \"paytm QR image URL\".
6. Collect payment using Paytm QR.
