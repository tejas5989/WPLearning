== Installation ==

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don't even need to leave your web browser. To do an automatic install of WooCommerce Enhanced Ecommerce Analytics Integration with Conversion Tracking, log in to your WordPress admin panel, navigate to the Plugins menu and click Add New.

In the search field type "WooCommerce Enhanced Ecommerce Analytics Integration with Conversion Tracking" and click Search Plugins. Once you have found our plugin you can install it by simply clicking Install Now. After clicking that link you will be asked if you are sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

= Installation Instructions =

1. For Ecommerce Tracking

    * Sign into your <a href="https://analytics.google.com/analytics/web/?authuser=0#provision/SignUp/">Google Analytics Account</a>
    * Enable Enhanced E-commerce for your profile/view. This is a profile / view level setting and can be accessed under Admin > View > E-commerce Settings
    * Add meaningful labels for your checkout steps. We recommend you to label as, 
		Step 1 : Checkout View; Step 2 : Billing Info; Step 3 : Proceed to payment
    * Activate our plugin from the Settings page. You can access the setting page from wordpress left tab DotStore Plugins >>WooCommerce Enhanced Ecommerce Analytics Integration With Conversion Tracking plugin
    * Find "Add Enhanced Ecommerce Tracking Code" in the settings page and check the box to add the tracking code
    * Click on Google Ecommerce Tracking Settings tab in the plugin.
    * Enable Google Ecommerce Tracking features in the plugin.
    * Add <a href="https://support.google.com/analytics/answer/1032385?hl=en">Google Analytics UID</a> in the plugin.

2. For Facebook conversion
	* Before Enable Facebook Conversion from our plugin Please follow steps and create "Facebook Track Id" from this link <a href="https://www.facebook.com/business/help/435189689870514?locale=en_GB" target="_blank">https://www.facebook.com/business/help/435189689870514?locale=en_GB</a> and enter "Facebook Track id" to our Plugin settings and boost your store marketing :)
3. For Google conversion
	* Before Enable Google Conversion from our plugin Please follow steps and create Google conversion from this link <a href="https://support.bigcommerce.com/articles/Public/How-do-I-add-Google-Adwords-Conversion-Tracking" target="_blank">https://support.bigcommerce.com/articles/Public/How-do-I-add-Google-Adwords-Conversion-Tracking</a> and enter "Google conversion id" and "Google conversion lable" to our plugin settings section then you are ready to boost your AdWords in your WooCommerce store.

== Frequently Asked Questions ==

= Q. How to Setup Ecommerce Tracking in your website? =
For set up Ecommerce tracking You have follow below list step 

* Sign into your <a href="https://analytics.google.com/analytics/web/?authuser=0#provision/SignUp/">Google Analytics Account</a>
* Enable Enhanced E-commerce for your profile/view. This is a profile / view level setting and can be accessed under Admin > View > E-commerce Settings
* Add meaningful labels for your checkout steps. We recommend you to label as,
  Step 1 : Checkout View; Step 2 : Billing Info; Step 3 : Proceed to payment
* Activate our plugin from the Settings page. You can access the setting page from wordpress left tab DotStore Plugins >>WooCommerce Enhanced Ecommerce Analytics Integration With Conversion Tracking plugin
* Find "Add Enhanced Ecommerce Tracking Code" in the settings page and check the box to add the tracking code
* Click on Google Ecommerce Tracking Settings tab in the plugin.
* Enable Google Ecommerce Tracking features in the plugin.
* Add <a href="https://support.google.com/analytics/answer/1032385?hl=en">Google Analytics UID</a> in the plugin.

= Q. How do I make sure Google Analytics is properly installed on all pages? =
If you signed it with google and selected your website (or manually added the tracking ID) the Google Analytics javascript will be added to all pages. just open any page of your website in Chrome, right click to select Inspect, navigate to Network tab, reload the page and search for google analytics, you will see the google code with your UA ID.(<a href="https://support.google.com/analytics/answer/1008083?hl=en#CheckRT">Check your web-tracking-code setup</a>)

= Q. I have installed the GA Google Analytics plugin and did insert the tracking code and also I checked in my website View Page Source the code is there, but in Google Analytic says tracking code is not detected. =
You need to wait awhile for Google to collect some data, like at least a day or whatever. Standard stuff for Google Analytics. For more information, check out the <a href="https://support.google.com/analytics/?hl=en#topic=3544906">Google Analytics Help Center</a>.

= Q. Do I need to copy/paste the Facebook Pixel code somewhere? =
No, you just have to enter your pixel ID in the dedicated field. The plugin will do the rest for you.

= Q. How to add the Facebook pixel code on every page of your WordPress site =
With the our plugin, you can install the Facebook Pixel code with just a single click. You don not even need the actual code, just the Pixel ID.  Once on your site, the pixel will start to send crucial data to Facebook right away. Paste the Facebook Pixel ID inside the dedicated field in the admin area of the plugin (be careful not to paste the full code of the pixel)

Note: Please follow this Guide to <a href="https://store.multidots.com/docs/plugins/woocommerce-enhanced-ecommerce-analytics-integration-conversion-tracking/facebook-conversion/">setup your Facebook Conversion Pixel</a>. 

= Q. How to get the Facebook Pixel ID =
Click on the Pixels linkYou will find a list with all your pixels and their IDs.

For more details about the pixel, you can click on the link. If you never used the pixel before you will be asked to set up the pixel. At this point simply add the pixel ID <a href="https://store.multidots.com/docs/wp-content/uploads/2016/12/pixel-id-1-1024x310.png">click here</a>

= Q. How to check facebook Pixel is added in website? =
Yes you can to test it, install the <a href="https://chrome.google.com/webstore/detail/facebook-pixel-helper/fdgfkebogiimcoedlicjlajpkdmockpc">free Chrome Pixel Helper extension</a> and then visit your site.<a href="https://store.multidots.com/docs/wp-content/uploads/2016/12/pixel-helper.png">click here</a>

= Q. How to test your facebook pixel in WooCommerce website? =

You have to follow below step:

Step 1: Install <a href="https://chrome.google.com/webstore/detail/facebook-pixel-helper/fdgfkebogiimcoedlicjlajpkdmockpc">Pixel Helper Chrome extension</a> in your Chrome browser

Step 2: Open your site within Chrome. In the top right corner you will see reports from the Pixel Helper extension. <a href="https://store.multidots.com/docs/wp-content/uploads/2016/12/helper-rep.png">click here</a>

Step 3: Click on that icon for details about the plugin. This will open a detail report with useful information

<strong>Important:</strong> Delete page cache. Make sure you are not testing the site with a user role that is excluded because, in this case, the pixel code will not be visible.

<strong>False Error:</strong> If  WooCommerce setup is activated you might see an error about the "AddToCart" event. This is because of the dynamic AddToCart event and will disappear after you click on the add to cart button. This error is normal and you can safely ignore it.
